Pertanyaan Refleksi
Setelah menyelesaikan kode, jawab pertanyaan berikut:

1. Apa fungsi keyword 'extends' dalam deklarasi class Java?
2. Jelaskan perbedaan antara Method Overloading (yang ada di latihan sebelumnya) dengan Method Overriding (yang ada di latihan ini)!
3. Apa kegunaan keyword 'super' di dalam constructor subclass (Manajer)?
4. Jika variabel di class Karyawan diubah access modifier-nya menjadi 'private', apa yang akan terjadi pada class Manajer? Bagaimana solusinya?
5. Mengapa kita perlu menggunakan Inheritance? Apa keuntungan utamanya dibandingkan membuat dua class yang terpisah sama sekali (misal class Karyawan dan class Manajer dibuat terpisah tanpa hubungan)?

Jawaban:
1.extends digunakan untuk membuat sebuah class mewarisi atribut dan method dari class lain.
  Artinya subclass otomatis mendapatkan semua properti superclass.
  contoh :
  class Manajer extends Karyawan
2.Overloading
  - Terjadi dalam satu class
  - Nama method sama, parameter berbeda
  - Tujuannya memberikan variasi penggunaan
  Overriding
  - Terjadi pada hubungan parent–child
  - Nama method sama, parameter sama
  - Tujuannya mengubah perilaku method parent
3.super(...) digunakan untuk:
  - memanggil constructor milik superclass
  - mengisi atribut yang diwarisi dari parent
  contoh :
  super(nama, gajiPokok);
4.Masalah:
  Subclass tidak bisa mengakses langsung variabel private.
  Solusi:
  Gunakan getter/setter
  atau
  Ubah ke protected agar bisa diakses subclass.
  Yang benar dalam OOP → pakai getter/setter.
5.Keuntungan inheritance:
  - Menghindari duplikasi kode (code reuse)
  - Struktur lebih rapi dan mudah dikembangkan
  - Polimorfisme dapat digunakan
  - Lebih mudah maintenance
  - Hubungan antar class menjadi jelas
  Jika membuat dua class terpisah:
  - Banyak kode terulang
  - Susah diperluas
  - Polimorfisme tidak bisa digunakan
